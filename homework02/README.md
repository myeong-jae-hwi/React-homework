# 과제 2

## Custom Vite Template

React 학습 시작을 위한 사용자 정의 템플릿

저장소 파일 복사

```sh
degit myeong-jae-hwi/React-homework/homework02 <폴더_이름>
```

해당 파일로 이동

```sh
cd <폴더_이름>
```

필요한 것들 다운로드

```sh
pnpm install
```

## 실행 명령어 💬

개발 서버 구동 💻

```sh
pnpm dev
```

프로젝트 빌드 🌍

```sh
pnpm build
```

프로젝트 프리뷰 👀

```sh
pnpm preview
```
